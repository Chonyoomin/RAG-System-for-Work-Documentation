# Sample Data

This folder may contain **only** synthetic, fictional, or fully redacted sample documents intended for public demonstration.

Do not place any of the following here:

- Proprietary, company-internal, or customer documents
- Real identifiers (names, emails, account IDs, employee IDs)
- Internal screenshots or exports from real systems
- Files derived from private sources (OCR output, extracted text, embeddings)

If a document cannot be safely viewed by anyone on the public internet, it does not belong in this repository. See [../../docs/github-safety.md](../../docs/github-safety.md).
